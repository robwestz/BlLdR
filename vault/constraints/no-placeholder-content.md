# Constraint: No Placeholder Content

## Scope
Always. All files containing user-visible text or media.

## Prohibited Patterns

### ❌ Lorem ipsum
**Banned:** "Lorem ipsum dolor sit amet" or any Latin filler text
**Why:** Clients see it and lose confidence. It ships to production more often than you think.
**Instead:** Write real content. If content isn't ready, write realistic placeholder that matches the domain.

### ❌ Default names
**Banned:** "Your Company Name", "John Doe", "example@email.com", "123 Main Street"
**Why:** Looks unprofessional and often ships unnoticed.
**Instead:** Use project-specific realistic content derived from onboarding answers.

### ❌ Placeholder images
**Banned:** Gray boxes, "image placeholder" text, broken image icons, via.placeholder.com
**Why:** Makes the entire build look unfinished.
**Instead:** Use real images (Unsplash) or project-specific imagery. Set alt text.
