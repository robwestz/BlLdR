# Routine: Accessibility Audit

## When to Run
After completing any module with visual UI.

## Checklist
- [ ] Color contrast ≥ 4.5:1 for body text
- [ ] Color contrast ≥ 3:1 for large text (18px+ or 14px+ bold)
- [ ] All images have alt text (decorative images: alt="")
- [ ] All form inputs have associated <label> elements
- [ ] Tab key reaches every interactive element in logical order
- [ ] Focus indicator visible on all focusable elements
- [ ] No information conveyed by color alone
- [ ] Heading hierarchy is sequential (h1→h2→h3, no gaps)

## If Any Check Fails
Fix the specific accessibility issue. Re-run this checklist.

## Duration
5 minutes per page.
