# Memory: Website Projects

## Scars

### Scar: No contact form validation
**When:** Contact form module
**What happened:** Built the form without client-side validation
**Consequence:** Inbox filled with empty submissions, invalid emails, spam. Client was furious.
**Now we:** Every field validates on blur AND submit. Email format checked. Required fields enforced.

### Scar: No CTA hierarchy
**When:** Page layout
**What happened:** Every section had a button, all styled the same
**Consequence:** Users didn't know what was important. Click-through rate was near zero.
**Now we:** One primary CTA per page. Secondary actions use secondary/ghost styling.

## Insights

### Insight: Five-second test
**When:** Homepage design
**What worked:** We showed the homepage to three people for 5 seconds and asked "what is this site about?"
**Why:** If they couldn't answer, the messaging was wrong. Simple test, huge impact.
**Apply:** After building the hero section, apply the 5-second test mentally. Is the purpose obvious?

### Insight: Simple navigation
**When:** Navigation design
**What worked:** Max 5 top-level nav items, no dropdowns, mobile hamburger as full-screen overlay
**Why:** Users found everything within 2 clicks. Simplicity felt premium.
**Apply:** Max 5 nav items. No mega-menus. Mobile nav is full-screen overlay.
