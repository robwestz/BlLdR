# Memory: Web App Projects

## Scars

### Scar: Pages before auth
**When:** Authentication
**What happened:** Built all protected pages before auth worked
**Consequence:** Couldn't test properly. When auth was added, half the pages broke.
**Now we:** Auth module fully working before any protected page is built.

## Insights

### Insight: Dashboard as summary
**When:** User dashboard design
**What worked:** Dashboard showed summary cards linking to detailed sections
**Why:** Users got overview instantly and could drill into what mattered.
**Apply:** Dashboard is a hub, not a destination. Summary + links to depth.
